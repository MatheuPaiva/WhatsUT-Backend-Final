=====================================================
 Tutorial para Rodar a Aplicação WhatsUT
=====================================================

Este guia explica como configurar e executar o backend e o frontend da aplicação WhatsUT em seu ambiente de desenvolvimento.

--------------------
 Pré-requisitos
--------------------
Antes de começar, certifique-se de que você tem os seguintes softwares instalados em sua máquina:

1.  **Node.js**: Essencial para executar o ambiente JavaScript. É recomendado usar a versão 18 ou superior. Você pode baixá-lo em: https://nodejs.org/
2.  **PNPM**: O projeto utiliza o `pnpm` como gerenciador de pacotes. Se você não o tiver, pode instalá-lo globalmente após instalar o Node.js com o seguinte comando:
    
    npm install -g pnpm

--------------------
 Passo 1: Configurar e Rodar o Backend (Servidor)
--------------------

O backend é responsável por toda a lógica de negócio, autenticação e gerenciamento de dados.

1.  **Abra um terminal** (como o PowerShell, CMD, ou o terminal do seu editor de código).

2.  **Navegue até o diretório raiz do projeto**, onde está o arquivo `package.json` principal.
    
    cd caminho/para/seu/projeto/whatsut-backend-final

3.  **Instale as dependências** do backend. O pnpm irá baixar todas as bibliotecas necessárias (NestJS, Express, etc.).
    
    pnpm install

4.  **Inicie o servidor do backend** em modo de desenvolvimento. Este comando irá compilar e iniciar o servidor, e ele será reiniciado automaticamente sempre que você fizer uma alteração nos arquivos do backend.
    
    pnpm run start:dev

5.  Após a inicialização, você verá mensagens no terminal indicando que o servidor está rodando, geralmente na porta 3000. **Mantenha este terminal aberto.**

--------------------
 Passo 2: Configurar e Rodar o Frontend (Interface do Usuário)
--------------------

O frontend é a interface visual com a qual o usuário interage, construída em React.

1.  **Abra um NOVO terminal.** É importante manter o terminal do backend rodando separadamente.

2.  **Navegue até o diretório `frontend`** dentro do projeto.
    
    cd caminho/para/seu/projeto/whatsut-backend-final/frontend

3.  **Instale as dependências** do frontend. Este comando irá baixar as bibliotecas do React, MUI, etc.
    
    pnpm install

4.  **Inicie o servidor de desenvolvimento do frontend.**
    
    pnpm run dev

5.  O terminal irá mostrar uma URL local, geralmente `http://localhost:5173`.

--------------------
 Passo 3: Acessar a Aplicação
--------------------

1.  Abra seu navegador de internet (Chrome, Firefox, etc.).
2.  Acesse a URL fornecida pelo terminal do frontend (normalmente **http://localhost:5173**).
3.  Pronto! A página de login do WhatsUT deve aparecer, e a aplicação estará totalmente funcional, conectada ao seu backend local.
